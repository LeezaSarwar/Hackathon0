============================================================
ANTHROPIC API KEY SETUP - REQUIRED FOR AI DRAFT REPLIES
============================================================

STEP 1: Get Your API Key
-------------------------
1. Go to: https://console.anthropic.com/
2. Sign in with your account
3. Click "API Keys" in the left menu
4. Click "Create Key"
5. Copy the key (starts with "sk-ant-")

STEP 2: Add Key to .env File
-----------------------------
1. Open this file in Notepad:
   D:\LEEZA\HACKATHON 0\silver\.env

2. Find this line:
   ANTHROPIC_API_KEY=your_anthropic_api_key_here

3. Replace "your_anthropic_api_key_here" with your actual key:
   ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxxx

4. Save the file

STEP 3: Restart Watcher
------------------------
1. Close any running watcher (Ctrl+C in terminal)
2. Double-click: START_WHATSAPP_FRESH.bat
3. QR code will appear - YOU MUST SCAN IT!
4. Pick up phone, open WhatsApp
5. Tap 3 dots → Linked Devices → Link a Device
6. Scan QR code on computer screen
7. Wait for "LOGIN SUCCESSFUL!" message

STEP 4: Test It
---------------
1. After login success, ask sister to send:
   "urgent need help"
2. Wait 30 seconds
3. Check: AI_Employee_Vault\Needs_Action\
4. You'll see: whatsapp_[name]_[timestamp].md
5. Open it - AI draft reply will be inside!
